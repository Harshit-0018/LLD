# 🚗 Smart Parking Lot System (Low Level Design)

A **Low Level Design (LLD)** based **C++ project** that simulates a real-world **Smart Parking Lot System** using **Object-Oriented Programming (OOP)** principles and clean system design.

Built with **interview preparation**, **scalability**, and **real-world thinking** in mind 🚀

---

## 🧩 Problem Statement
Design a parking lot system that:
- 🏢 Supports multiple parking floors
- 🚙 Handles different vehicle types
- 🎯 Allocates parking slots efficiently
- 🎟️ Generates parking tickets
- 🧠 Follows OOP & SOLID principles

---

## ✨ Key Features
- 🚗 Vehicle-based slot allocation
- 🏬 Multi-floor parking support
- ⏱️ Ticket generation with entry timestamp
- ♻️ Singleton-based ParkingLot controller
- 🧩 Modular & extensible architecture

---

## 🛠️ Tech Stack
- 💻 Language: C++
- 🧠 Paradigm: Object-Oriented Programming (OOP)
- 🧩 Design Pattern: Singleton
- 🏗️ Concepts Used: Encapsulation, Abstraction, Modularity

---

## 🧱 Class Design Overview

### Core Components
- 🚘 Vehicle – Stores vehicle details and type  
- 🅿️ ParkingSlot – Represents an individual parking slot  
- 🏢 ParkingFloor – Manages slots on each floor  
- 🎫 Ticket – Stores parking session information  
- 🧠 ParkingLot – Central system controller (Singleton)
