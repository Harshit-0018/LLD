# Smart Parking Lot System (LLD)

## Overview
A Low Level Design project demonstrating OOP and SOLID principles using C++.

## Features
- Multiple floors
- Vehicle-based slot allocation
- Ticket generation
- Singleton ParkingLot

## How to Run
```
g++ src/*.cpp -o parking
./parking
```

## Concepts Used
- OOP
- Singleton Pattern
- Encapsulation
- Modularity

## Future Improvements
